2. Fundamental Concepts

Passive vs. Active Measurements

Aspect Passive Fixture Active Amplifier
Power Supply None Required (VCC, VGS/IDS)
Bias Point N/A Critical (defines gain, impedance)
S-Parameters Frequency-independent (ideally) Bias-dependent
Stability Always stable Risk of oscillation
Measurement Concern Path loss, discontinuities Gain, impedance transformation, oscillation

Key Terminology

  • S-parameters: Scattering parameters; voltage ratios in a 50 Ω system
  • Z-parameters: Impedance matrix; includes transimpedance (V/I)
  • Return loss: dB magnitude of reflected signal; RL = −20 log₁₀(|Γ|)
  • Insertion loss: dB magnitude of transmitted signal through passive network
  • Bias tee: High-pass network for RF, low-pass for DC; allows simultaneous AC/DC injection
  • Transimpedance (TIA): Amplifier topology converting input current to output voltage
  • De-embedding: Mathematically removing fixture parasitics to move reference plane to DUT terminals

3. dBm to Watts & Voltage Conversion

Conversion Formulas

Pwatts = 10(dBm/10) / 1000
Vpeak (50Ω) = √(Pwatts × 50)

Reference Table: 10–40 dBm

dBm Watts mW Peak Voltage (V) Peak Voltage (mV)
10 0.01 W 10 mW 0.707 V 707 mV
20 0.1 W 100 mW 2.236 V 2,236 mV
30 1 W 1,000 mW 7.071 V 7,071 mV
40 10 W 10,000 mW 22.361 V 22,361 mV

Key Insight: Decibel Doubling Rule

  • Every +10 dB = 10× power increase
    • 10 → 20 dBm: 10 mW → 100 mW
    • 20 → 30 dBm: 100 mW → 1 W
    • 30 → 40 dBm: 1 W → 10 W
  • Every +3 dB ≈ 1.414× voltage increase (√2)

VNA Safety Context

Power Status Recommendation
+10 to +20 dBm Safe Typical small-signal drive
+20 to +25 dBm Marginal Add attenuator if unsure
+25 to +30 dBm At limit Use 6–10 dB attenuator
>+30 dBm Exceeds rating Mandatory 10+ dB attenuator (damage risk)

4. VNA Measurement for Active Amplifiers

System Architecture

┌─────────────────────────────────────┐ │ AMPLIFIER UNDER TEST │ │ │ Port 1 ────→│ [Bias Tee 1] Amp [Bias Tee 2] │────→ Port 2 (50Ω) │ ↓ ↓ │ (50Ω) │ VCC VCC │ │ │ │ S11 S21 S22 │ │ S12 (gain) (out) │ └─────────────────────────────────────┘

Calibration Strategies

Option 1: SOL at VNA Ports (Standard)

Simple 50 Ω calibration at the VNA connectors, before bias tees. Fastest approach; bias tees remain in measurement.

Advantages:
  • Simple; uses standard 50 Ω calibration kit
  • Bias tees are passive; included in measurement
  • Fast calibration (~5 minutes)
Disadvantages:
  • Bias tee parasitics remain in measurement
  • If tees are lossy, dynamic range reduced

Option 2: TRL at Bias Tee Plane (Advanced)

Thru-Reflect-Line calibration at bias tee output. More complex but best accuracy; automatically de-embeds bias tees.

Advantages:
  • De-embeds bias tee insertion loss & phase
  • Impedance-agnostic (works even if not 50 Ω)
  • Best accuracy for active devices

Measurement Procedure

  1. Calibrate VNA: SOL (quick) or TRL (better)
  2. Apply DC Bias: Set VCC, gate/collector bias; monitor with DMM
  3. Verify Stability: Check with spectrum analyzer; look for oscillation
  4. Sweep Frequency: Typically 100 MHz to 40–50 GHz
  5. Capture All S-Parameters: S₁₁, S₁₂, S₂₁, S₂₂
  6. Export Touchstone (.s2p): Document bias point and conditions

De-embedding: Moving Reference Planes

Port Extension Method (Simplest)

Removes matched transmission line effect from measurement plane to DUT terminals.

τ = L / vp

where L = physical length, vp = phase velocity (~0.6c on PCB)

Fixture De-embedding (Advanced)

  1. Measure fixture alone → S_fixture
  2. Measure fixture + DUT → S_total
  3. Remove cascade using ABCD algebra → S_DUT

Impedance Extraction from S-Parameters

Zin = Z₀ × (1 + S₁₁) / (1 − S₁₁) = 50 Ω × (1 + S₁₁) / (1 − S₁₁)
Zout = Z₀ × (1 + S₂₂) / (1 − S₂₂)

Gain & Stability Analysis

G = 20 × log₁₀(|S₂₁|) dB
K = (1 + |Δ|² − |S₁₁|² − |S₂₂|²) / (2 × |S₁₂ × S₂₁|)

K > 1: Unconditionally stable | K < 1: Potentially unstable

5. Bias Tees & Conical Coils

What is a Bias Tee?

│ VCC (DC) │ R (10-100Ω) │ ┌────┴────┬─────→ RF_in/RF_out │ │ [L] [C_block] (RF) (AC coupling) │ │ └────┬────┘ │ GND

Function:

  • Capacitor: Blocks DC from RF path; passes AC
  • Inductor: Blocks RF from DC supply; passes DC

Conical Coil Design

A conical spiral (tapered diameter) avoids sharp resonances by distributing capacitance and inductance across frequency:

Tip (small diameter) ▲ ╱│╲ ╱ │ ╲ ← High freq: small turns (low inductance) ╱ │ ╲ ╱ │ ╲ ╱ │ ╲ ╱ │ ╲ Base (large diameter) ↑ Low freq: large turns (high inductance)

6. Output Attenuators & VNA Protection

Why Output Attenuators Are Critical

VNA Receiver Maximum Input Power:
  • Keysight N5230C: +30 dBm
  • Rohde & Schwarz ZNA: +25 dBm
Exceeding this limit → burns out low-noise input stage (permanent damage)

Calculation Example

Safe Scenario:
  • Amplifier gain: +30 dB
  • VNA Port 1 output: +5 dBm
  • Amplifier output: 5 + 30 = +35 dBm (exceeds limit)
  • Solution: Add 10 dB pad → −5 dBm to VNA ✓ Safe

Attenuator Selection Table

Amp Gain Input Power Amp Output Attenuator Final to VNA Status
+15 dB −5 dBm +10 dBm None +10 dBm ✓ Safe
+25 dB +0 dBm +25 dBm 6 dB +19 dBm ✓ Safe
+30 dB +5 dBm +35 dBm 10 dB +25 dBm ✓ Safe
+40 dB +0 dBm +40 dBm 20 dB +20 dBm ✓ Safe

De-embedding the Attenuator

The attenuator is now part of your measured S-parameters. You can either:

  • Option A: Report the cascade ("S-parameters measured with 10 dB output pad")
  • Option B: De-embed the attenuator using cascade removal (ABCD matrix inversion)

7. TIA Measurements with VNA

Challenge: High-Impedance Input

Transimpedance Amplifier (TIA): Converts input current to output voltage

  • Input impedance: 25–100 Ω (typically set by feedback resistor)
  • BUT: Without photodiode, input appears complex (capacitive)
  • VNA is 50 Ω system → potential impedance mismatch

Methodology 1: Direct S→Z Conversion ✓ Most Common

VNA Port 1 ──[DC block]──→ TIA input ↓ [Bias tee] ↓ VNA Port 2 ←───────── TIA output

Procedure:

  1. Calibrate VNA at 50 Ω (SOL)
  2. Apply DC bias (~1 mA typical for op-amp TIA)
  3. Measure S-parameters (−30 to −40 dBm input power, 100 kHz–10 GHz)
  4. Convert S → Z using matrix transformation
  5. Extract Z₂₁ (transimpedance gain in Ohms)
Z₂₁ = Z₀ × 2S₂₁ / ((1−S₁₁)(1−S₂₂) − S₁₂S₂₁)
Z₂₁(dBΩ) = 20 × log₁₀(|Z₂₁|)

Methodology 2: Impedance Matching Network

For very high source impedance (>1 MΩ photodiode):

Photodiode ──[L: 50-200nH]──[Transformer 16:1]──→ TIA (50Ω) (high Z, µA) (series) (impedance ratio) (low Z)

Methodology 3: Transmission Line Matching

PCB traces between photodiode and TIA matched to TIA input impedance (±50%):

  • TIA Z₁ = 100 Ω → trace impedance 50–150 Ω
  • Minimizes reflections at Gbps data rates
  • Integrated design approach

Critical Factor: Photodiode Capacitance

f₋₃dB ≈ gm / (2π × (CPD + Copamp))
Key Insight: When measuring TIA without photodiode, bandwidth is artificially high. Add 10–100 pF capacitor to simulate real photodiode for accurate results.

8. Practical Examples & Literature

Example 1: 10 Gbps Optical TIA (Microwave Journal, 2012)

Reference: "Fast Calculation of Transimpedance Gain and Equivalent Input Noise Current Density for High-speed Optical Preamplifier Design"

  • Feedback resistance: 5 kΩ (variable)
  • Frequency range: 100 kHz–10 GHz (VNA measurement)
  • Peak Z₂₁: ~80 dBΩ (5000 Ω)
  • Photodiode capacitance dependency:
    • C_PD = 0.1 pF → BW ≈ 14 GHz
    • C_PD = 0.6 pF → BW ≈ 8 GHz (10 Gbps limit)
    • C_PD > 1 pF → BW < 5 GHz (unsuitable)
Microwave Journal: Fast Calculation of Transimpedance Gain

Example 2: 133 GHz InP DHBT TIA (UCL, 2018)

Reference: "Transimpedance Amplifiers with 133 GHz bandwidth on 130nm InP DHBT"

  • Measurement: On-wafer with Cascade Infinity 67 GHz GSG probes
  • Input impedance (Z₁₁): 25–35 Ω (well-matched)
  • Transimpedance gain (Z₂₁): 42 dBΩ across 0–133 GHz
  • Data rate tested: 64 Gbps eye diagrams
  • Significance: Highest bandwidth TIA at publication
UCL Discovery: 133 GHz Bandwidth TIAs

Example 3: OPA818 TIA Testing (TI E2E Forum, 2021)

Problem: Measured gain lower than expected (74 dBΩ) without photodiode attached

Root Cause: Unmeasured input capacitance (op-amp parasitic + PCB)

Solution: Added 33 pF capacitor in parallel with feedback resistor to simulate photodiode; gain then matched predictions

Lesson: Always simulate input capacitance when measuring TIA without photodiode.

Summary: Key Takeaways

  1. dBm conversions matter: Know the power at each stage
  2. Bias tees are critical: Use 4310LC for DC–6 GHz, BCR for broadband to 40 GHz
  3. Output attenuators protect VNA: Keep receiver input <+20 dBm
  4. De-embedding is optional but useful: Extract device-only specs post-measurement
  5. TIA measurements require S→Z conversion: Direct measurement, convert to Z-parameters
  6. Photodiode capacitance dominates TIA bandwidth: Simulate with realistic input capacitance (~0.6–1 pF)
  7. Stability first: Check for oscillation before connecting to VNA
  8. Calibration with bias tees in-place: TRL calibration is cleanest approach
